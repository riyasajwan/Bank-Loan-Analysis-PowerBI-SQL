# 🏦 Bank Loan Analysis Dashboard using MySQL & Power BI

An end-to-end **Bank Loan Analysis Dashboard** built using **MySQL** and **Power BI** to analyze loan performance, funding trends, repayment behavior, and key business KPIs. This project demonstrates SQL querying, data analysis, DAX calculations, and interactive dashboard development.

---

## 📌 Project Overview

The objective of this project is to analyze historical bank loan data and build an interactive dashboard that helps stakeholders monitor loan performance and make informed business decisions.

The project covers:

- Loan application trends
- Funded amount analysis
- Total amount received
- Good vs Bad loan analysis
- State-wise loan distribution
- Loan purpose analysis
- Home ownership analysis
- Monthly performance trends

---

## 🎯 Business Problem

Banks receive thousands of loan applications every year. Understanding borrower behavior and loan performance is essential to:

- Improve lending decisions
- Monitor repayment performance
- Identify high-risk loans
- Track funding and revenue trends
- Support data-driven business decisions

This dashboard provides insights into these business questions through interactive visualizations.

# 💡 Business Insights

Some key insights generated from this analysis include:

- Most loans fall under the **Good Loan** category.
- Monthly loan applications show seasonal trends.
- Mortgage and Rent are the dominant home ownership categories.
- Debt Consolidation is one of the most common loan purposes.
- Certain states contribute significantly more loan applications than others.
- Good loans account for the majority of funded and recovered amounts.
